# Assets

This directory contains visual assets for the game.

## Subdirectories

- **sprites/**: Character and entity sprites
- **tiles/**: Tile graphics for the dungeon map
- **ui/**: UI elements and icons

## Current Implementation

The game currently uses canvas-based rendering with simple geometric shapes:
- Player: Red circle
- Rooms: Colored rectangles
- Boss rooms: Red rectangles with crown emoji
- Completed rooms: Green rectangles with checkmark

## Future Enhancements

You can add custom sprite sheets and tile graphics to enhance the visual appearance of the game.
